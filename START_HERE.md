## 🚀 3 Simple Steps to Deploy

### Step 1️⃣: Create GitHub Account (2 minutes)
1. Go to [github.com/signup](https://github.com/signup)
2. Enter your email, create password
3. Verify your email
4. Done! ✅

### Step 2️⃣: Upload Code to GitHub (5 minutes)
1. Log in to GitHub
2. Click **"+"** (top right) → **"New repository"**
3. Name it: `sarisari-pos`
4. Check **"Initialize with README"**
5. Click **"Create repository"**
6. Click **"Add file"** → **"Upload files"**
7. **Drag ALL files from this folder** and upload
8. Click **"Commit changes"**

### Step 3️⃣: Deploy to Vercel (3 minutes)
1. Go to [vercel.com](https://vercel.com)
2. Click **"Sign in with GitHub"**
3. Click **"Add New..."** → **"Project"**
4. Select your `sarisari-pos` repository
5. Click **"Deploy"**
6. Wait 2-3 minutes... ⏳
7. **DONE!** 🎉 Your website is live!

## 🎥 Camera Scanning

Once deployed, the **Scan button** will work!

**To test camera:**
1. Open your deployed website
2. Click **"Scan"** button
3. Browser asks for camera permission
4. Click **"Allow"**
5. Point at QR code
6. Product added automatically! ✨

**Generate QR codes:**
- Go to: https://www.figma.com/qr-code-generator/
- Type product ID (1, 2, 3... up to 10)
- Download & print QR code
